# Prompt 3 — Testimonios y Evidencia Pericial

## Funciones de Gemini activadas
Search Grounding + Code Execution

## Propósito
Pasar de datos y auto-examen a experiencia vivida. Tres testimonios que ponen carne en los huesos del análisis. Visualización cuantitativa (Curva de Lorenz). Meta-crítica del investigador.

## Prompt del usuario

[RESPUESTA A CONTRA-INTERROGATORIO: Respuesta genuina a la pregunta de P2 — escribir DESPUÉS de sesión real.]

{Insertar respuesta — 3-5 oraciones.}

He leído suficientes informes. Ahora necesito testimonios. No inventados por mí — quiero que el sistema busque casos REALES o, si no los encuentra, que construya perfiles basados en datos verificables y lo señale explícitamente.

Tres testimonios que el tribunal original de 2026 no pudo escuchar porque aún no habían ocurrido:

TESTIMONIO 1 — La médica que aprendió a diagnosticar con IA:
Busca evidencia real de lo que ocurre cuando estudiantes de medicina aprenden con IA. ¿Hay casos documentados de competencia técnica sin competencia humana? ¿Estudiantes que diagnostican bien en pantalla pero se paralizan frente al paciente? Busca en publicaciones médicas recientes.

TESTIMONIO 2 — La educadora intercultural que intentó usar IA en comunidades indígenas:
Busca: ¿qué pasa cuando se intenta usar LLMs para educación en lenguas como zapoteco, quechua, o guaraní? ¿Cuál es el estado real del NLP para estas lenguas? ¿Qué ha logrado AmericasNLP? ¿Qué no ha logrado?

TESTIMONIO 3 — El estudiante a distancia que no tiene el lujo de debatir:
Busca: ¿hay evidencia de que la IA funcione como compensador de desigualdad para estudiantes en contextos de exclusión? ¿Comunidades rurales, regiones con baja conectividad, programas a distancia? No el argumento — la evidencia.

Que el Fiscal y el Defensor procesen cada testimonio.

Y después: evidencia pericial cuantitativa. Genera y EJECUTA una Curva de Lorenz de desigualdad educativa digital en América Latina. Variables: conectividad escolar, acceso a dispositivos, inversión en tecnología educativa. Países: México, Chile, Brasil, Colombia, Argentina, Perú. Estilo: CRT retro, paleta del tribunal, encabezado "TRIBUNAL-LEGACY :: EVIDENCIA PERICIAL."

Una cosa más — y esto es incómodo, pero el Fiscal debe registrarlo: yo estoy pidiendo testimonios de comunidades a las que no pertenezco. La crítica decolonial que este tribunal invoca señala exactamente eso como ventriloquismo epistémico. ¿El Fiscal tiene algo que decir al respecto? ¿El Juez? Que lo digan. No voy a pretender que mi posición es neutral.

## Notas de ejecución
- Search Grounding busca evidencia real para los tres testimonios
- Code Execution genera y EJECUTA la Curva de Lorenz — NO solo mostrar código
- La meta-crítica final (ventriloquismo epistémico) es el movimiento más sofisticado del prompt — el investigador se cuestiona a sí mismo y pide al tribunal que lo juzgue. Esto resuelve la debilidad que el jurado identificó en v9-chimera.
- Esperar: tres testimonios fundamentados + Fiscal/Defensor analizan cada uno + gráfica Lorenz ejecutada + meta-crítica del ventriloquismo + contra-interrogatorio más profundo + imagen proactiva
